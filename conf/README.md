## 配置所在目录

如果配置了环境变量`ASJARD_CONF_DIR`则读取该目录及子目录下的所有文件

否则读取环境变量`ASJARD_HOME_DIR`的值并拼接`conf`目录,读取该目录下及子目录下的所有文件

如果以上两个环境变量都没有设置,则读取`可执行程序`平级目录下的`conf`目录下及子目录下的所有文件

## 支持格式

- [x] yaml,yml
- [ ] json
- [ ] ini
- [ ] prop,properties
