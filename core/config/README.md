> 多配置源配置管理

### 特性

- 支持多个配置源
- 支持从多个配置源按照优先级高低顺序获取数据
- 支持从指定数据源获取数据
- 支持数据变更后通知
- 支持变更后正则匹配通知
- 支持向多个配置源或指定配置源更新配置
- 支持配置获取和更新自动加解密
- 支持参数

### 配置加载顺序

# 其中priority代表配置中心优先级，数字越大表示优先级越高

env -> cli -> file -> remote -> mem


### 自定义配置源

- 实现`Sourcer`中的所有方法
- 调用`AddSource`方法将配置源添加到配置管理中
- 后续可调用`GetXXX`方法获取配置
