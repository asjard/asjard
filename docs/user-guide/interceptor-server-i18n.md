## 拦截器名称

i18n

## 支持协议

- HTTP

## 功能

- 错误信息国际化返回

## 配置

- 国际化配置存放路径
  - 优先读取`ASJARD_I18N_DIR`环境变量
  - 如果为空读取`ASJARD_HOME_DIR`
  - 如果为空则为`可执行程序`所在路径的`locals`目录
- 文件格式为`{lang}.json`， 其中`lang`为语言
- lang从请求头`lang`中获取(暂未支持Accept-Language请求头)

文件内容格式为:

```json
{
  "123": {
    "prompt": "错误提示"
    "doc": "可以自行处理这个错误的文档地址"
  }
}
```

其中`123`为错误码, 只包含`系统码`和`错误码`
